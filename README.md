Meat Cafe Management System

Meat Cafe - це консольна програма для керування замовленнями в кафе. 

Основний функціонал:
- Перегляд меню: Вибір страв та напоїв.
- Управління замовленням: Додавання, видалення, перегляд позицій та очищення замовлення.
- Робота з файлами: Збереження та завантаження замовлень за номером.
- Розрахунок чеку: Підрахунок загальної суми, податку (GST) та чайових(у процентах та у ручному форматі вводу).
- Режими обслуговування: Вибір між Takeaway та Dine-in, вибір свого номеру рахунку.

Як використовувати:
1. Запустіть програму.
2. Введіть номер замовлення.
3. Обирайте пункти меню через головне меню програми.
